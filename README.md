# 📊 Vishwajeet Power BI Dashboard

![Dashboard Banner](Template%20dark-gradient.jpg)

## 🚀 Project Overview

This project contains an interactive **Power BI Ecommerce Dashboard** built to analyze and visualize ecommerce sales performance using real-world business data.

The dashboard provides insights into:

- 📦 Sales Performance
- 💰 Profit Analysis
- 🛒 Order Trends
- 🌍 State-wise Sales
- 👥 Customer Insights
- 📈 Monthly Growth Analysis
- 🧾 Category & Sub-Category Performance
